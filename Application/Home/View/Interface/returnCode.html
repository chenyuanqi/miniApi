返回码说明

编号  :   00001

基础支持 　 描述：使用须知
No.1 token 除登录、注册外, 其他都要带上它 (GET 方式);
No.2 token 每次使用后刷新其最后有效时间; 重新登录或异地登录则更改 token
No.3 每次调用接口时, 可能获得正确或错误的返回码，开发者可以根据返回码信息调试接口, 排查错误。

参照表

返回值 简要说明
errcode 系统返回码，00001 代表成功，00002 代表失败
errinfo 系统返回信息
